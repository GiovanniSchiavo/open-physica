---
title: Cards
description: Griglie di schede per navigazione
---

Il componente `<Cards>` crea una griglia di schede cliccabili (`<Card>`). È
molto utile per creare indici visivi o collegamenti a sezioni correlate.

## Esempio

<Tabs items={["Risultato", "Codice"]}>
  <Tab value="Risultato">
    <Cards>
      <Card title="Cinematica" href="#">
        Studio del moto dei corpi senza considerarne le cause.
      </Card>
      <Card title="Dinamica" href="#">
        Studio delle forze e delle cause del moto.
      </Card>
    </Cards>
  </Tab>
  <Tab value="Codice">
    ```mdx
    <Cards>
      <Card title="Cinematica" href="#">
        Studio del moto dei corpi senza considerarne le cause.
      </Card>
      <Card title="Dinamica" href="#">
        Studio delle forze e delle cause del moto.
      </Card>
    </Cards>
    ```
  </Tab>
</Tabs>
