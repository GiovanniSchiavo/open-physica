---
title: Files (File Tree)
description: Visualizzazione struttura directory
---

Il componente `<Files>` permette di visualizzare la struttura di file e cartelle
in modo grafico. Utile per spiegare l'organizzazione di un progetto o dove
salvare file.

## Esempio

<Tabs items={["Risultato", "Codice"]}>
  <Tab value="Risultato">
    <Files>
      <Folder name="content" defaultOpen>
        <Folder name="docs">
          <File name="index.mdx" />
          <File name="guide.mdx" />
        </Folder>
        <File name="meta.json" />
      </Folder>
      <File name="package.json" />
    </Files>
  </Tab>
  <Tab value="Codice">
    ```mdx
    <Files>
      <Folder name="content" defaultOpen>
        <Folder name="docs">
          <File name="index.mdx" />
          <File name="guide.mdx" />
        </Folder>
        <File name="meta.json" />
      </Folder>
      <File name="package.json" />
    </Files>
    ```
  </Tab>
</Tabs>
